# People - Person Recognition Tool

Ferramenta de visão computacional para indicar quais períodos de um vídeo possuem a presença de pessoas.

## Compilação

1. Importe o projeto no Eclipse.
2. Clique com o botão direito do mouse sobre ele.
3. Selecione a opção exporte.
4. Selecione a opção "Runnable JAR file" e avançe.
5. Defina o caminho de saída e a configuração de execução (que deve usar a classe br.imd.vision.Main).
